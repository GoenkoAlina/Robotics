1. cat - вывод содержимого файла.
2. touch - создание нового файла / обновление существующего файла.
3. history - вывод истории терминальных команд.
4. pwd - вывод текущей директории.
5. mkdir - создание каталога.
6. free - вывод информации о памяти системы.
7. top - вывод информации о процессах.
8. alias - присвоение командам других названий.
9. sudo - выполнение команд с правами суперпользователя.
10. wget - скачивание файлов по URL в указанную папку.
